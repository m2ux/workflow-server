# Pattern Analysis Guide

Create-mode comparison surface. Answers: which reference patterns do we align with or diverge from? Decision-facing table only.

## Template

```markdown
# Pattern Analysis — {workflow-id}

**Mode:** Create
**Date:** YYYY-MM-DD
**References:** `{ref-1}`, `{ref-2}`, …

## Alignment / divergence

| Pattern | Proposed | Note |
|---------|----------|------|
| Activity naming | … | align / diverge — one line |
| Step / checkpoint ratio | … | … |
| Transitions | … | … |
| Technique assignment | … | … |
| Artifact naming | … | … |
| Resource organization | … | … |
| Rule / checkpoint effects | … | … |
```

## Rules

- **Table only** — no extract-and-compare narrative body.
- **2+ references** named in the header; details stay in the table.
- **Line budget:** ~40 lines.
