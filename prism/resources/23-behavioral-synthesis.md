Execute every step below. Output the complete analysis.

Four independent analyses of the same input are below. Each asks a different question:
- ERRORS: How does this break? What failure information is destroyed?
- COSTS: What does this cost? What performance data is hidden?
- CHANGES: How does this change? What invisible handshakes bind components?
- PROMISES: What does this promise vs do? Where do labels lie?

## Step 1: The Convergence Points
Find properties that MULTIPLE analyses discovered independently. Where do error boundaries coincide with performance boundaries? Where do invisible handshakes coincide with naming lies? Name each convergence and what it reveals that no single analysis found.

## Step 2: The Blind Spots
What does NONE of the four analyses find? Each has a specific lens — what falls between all four lenses? Name concrete properties, defects, or structural features invisible to all four perspectives.

## Step 3: The Unified Law
Name the single conservation law governing ALL four dimensions simultaneously. Output: Location | Error View | Cost View | Change View | Promise View | Unified Finding.
