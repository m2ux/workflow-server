## Capability

Initialize a high-sensitivity security fix: configure the private `security` remote, create the local security feature branch off the private fork, set up an isolated planning folder, surface a plain-language risk overview, and confirm the public `origin` remote is not tracked. This group decomposes the setup into ordered operations: [configure-security-remote](./configure-security-remote.md), [initialize-security-branch](./initialize-security-branch.md), [initialize-planning-folder](./initialize-planning-folder.md), [present-security-overview](./present-security-overview.md), and [verify-origin-untracked](./verify-origin-untracked.md).
